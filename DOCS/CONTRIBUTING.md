# Contributing to Astral

Thank you for your interest in contributing to Astaraxia's software!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a feature branch

## Requirements

- All code must be GPL-3.0-only compatible
- Use SPDX license identifiers
- Follow CODING_STYLE.md

## Pull Request Process

1. Ensure all tests pass
2. Update documentation if needed
3. Add yourself to AUTHORS (if not already there)
4. Submit PR with clear description
5. Wait 2 whole weeks for Me to review your PR

## Code Review

- All submissions require review
- Address feedback promptly
- Keep changes focused and atomic

## Licensing

By contributing, you agree that your contributions will be licensed under GPL-3.0-only.

## Developer Certificate of Origin (DCO)

All commits must include a Signed-off-by tag. See [AGENTS.md](../AGENTS.md) for details.
